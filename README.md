            - 스네이크케이싱

    - 파이썬연산
        * 기본연산자
            - 4대 연산자 : +-*/
            - 나머지 연산자 %
            - 몫 연산자 //
            - 제곱 연산자 **
            - 비교 연산자(6) > >= < <= != ==
            - 논리 연산자(3) and or not
        
        * 복합연산자
            = 대입연산자
            +=

        * in 연산자
